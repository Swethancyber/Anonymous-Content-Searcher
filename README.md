# Anonymous Content Searcher

## Description
Anonymous Content Searcher Is A Simple Python Program That Uses  **pywhatkit and time**

## Libraries Used
### 1. pywhatkit
- Sends A Request To The Web Browser And Open A New Tab
- Searches For The Relevant Content That You Provide In The Input Field

  ## 2. Time
  - Used To Calculate The **execution time** Of The Program And Determine How Many Seconds It Takes To Open A Tab And Get The Content

  ## Simple About This Program

  This Program Allows The User Enter Any Content, And It Will Open The Web Browser To Search And Display It
